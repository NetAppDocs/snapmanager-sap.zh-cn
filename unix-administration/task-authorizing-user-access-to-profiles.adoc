= 授权对配置文件的访问
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
除了基于角色的访问控制（ Role-Based Access Control ， RBAC ）之外，您还可以通过 SnapManager 为配置文件设置密码，以防止未经授权的访问。

.步骤
. 输入以下命令：
+
`* smsap凭据集-profile -name _profile_name_[-password _password_]*`
