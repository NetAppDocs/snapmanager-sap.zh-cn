= 执行管理操作
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以在设置和配置 SnapManager 后执行管理任务。通过这些任务，您可以管理备份，还原和克隆以外的正常操作。

管理员可以使用图形用户界面或命令行界面执行操作。
