= 存储自定义脚本的加密密码
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
默认情况下， SnapManager 不会将主机凭据存储在用户凭据缓存中。但是，您可以更改此设置。您可以编辑`smsap.config`文件以允许存储主机凭据。

`smsap.config`文件位于`<default installation location>/properties/smsap.config`

. 编辑`smsap.config`文件。
. 将` host.credentials.persistent_`设置为`* true *`。
