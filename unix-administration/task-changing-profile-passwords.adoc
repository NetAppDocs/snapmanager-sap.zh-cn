= 更改配置文件密码
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
要保护存储库中的现有配置文件，您应更新这些配置文件的密码。您可以在使用此配置文件创建备份时应用此更新的密码。

.步骤
. 要更新现有配置文件的配置文件密码，请输入以下命令：
+
`* smsap profile update -profile _profile_name_-profile-_password password_*`
