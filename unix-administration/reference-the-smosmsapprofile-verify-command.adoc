= smsap profile verify命令
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以运行 profile verify 命令来验证配置文件设置。在运行此命令之前，必须挂载数据库。



== 语法

[listing, subs="+macros"]
----
pass:quotes[smsap profile verify
-profile _profile_name_
[-quiet | -verbose\]]
----


== Parameters

* ``*-配置文件*``
+
指定要验证的配置文件。配置文件包含数据库的标识符和其他数据库信息。

* ``*-静默*``
+
在控制台中仅显示错误消息。默认情况下会显示错误和警告消息。

* ``*-详细*``
+
在控制台中显示错误，警告和信息性消息。
