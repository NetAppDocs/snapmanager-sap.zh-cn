= 删除单个资源的凭据
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以删除任何一个受保护资源的凭据，例如配置文件，存储库或主机。这样，您就可以仅删除一个资源的凭据，而不是清除所有资源的用户凭据。
