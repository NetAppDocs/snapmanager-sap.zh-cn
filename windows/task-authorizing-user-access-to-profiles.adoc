= 授权对配置文件的访问
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用 SnapManager 可以为配置文件设置密码，以防止未经授权的访问。

.步骤
. 输入以下命令：
+
`smsap credential set -profile -name profile_name [-password password]`
