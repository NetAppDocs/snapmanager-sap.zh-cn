= 查看备份列表
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用`smsap backup list`命令检查为配置文件创建的备份以及备份状态。对于每个配置文件，命令首先显示有关最新备份的信息，然后继续显示所有备份的信息。

.步骤
. 输入以下命令：
+
`* smsap备份列表-profile _profile_name_[-elibreer _character_][data _-archivelogs][-quiet _-verbes]*`
