= 查看与配置文件关联的特定操作的详细历史记录
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以查看与配置文件关联的特定 SnapManager 操作的详细历史记录。

.步骤
. 要显示与配置文件关联的特定SnapManager 操作的详细历史记录信息、请输入以下命令：
+
`smsap history operation-show -profile profile_name｛-label label |-id id｝｛-quiet |-verbose｝`
