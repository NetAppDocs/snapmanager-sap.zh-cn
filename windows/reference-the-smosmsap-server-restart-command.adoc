= smsap_server restart命令
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
此命令将重新启动 SnapManager 主机服务器，并以 root 身份输入。



== 语法

[listing]
----
smsap_server restart
[-quiet | -verbose]
----


== Parameters

* *`-静默`*
+
指定控制台上仅显示错误消息。默认情况下会显示错误和警告消息。

* *`-详细`*
+
指定控制台上显示错误，警告和信息性消息。





== 命令示例

以下示例将重新启动主机服务器。

[listing]
----
smsap_server restart
----
