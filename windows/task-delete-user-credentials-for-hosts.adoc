= 删除主机的用户凭据
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以删除主机的凭据，使用户无法再访问它。使用此命令，您可以仅删除一个资源的凭据，而不是清除所有资源的所有用户凭据。

.步骤
. 要删除用户的主机凭据、请输入以下命令：
+
`smsap credential delete -host -name_host_name_-username_-username_`
