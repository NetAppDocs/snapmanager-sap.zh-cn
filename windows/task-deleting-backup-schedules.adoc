= 删除备份计划
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以删除不再需要的备份计划。

. 要删除备份计划、请输入以下命令：`smsap schedule delete -profile profile_name-schedule-nameschedulename (-quiet |-verbose)`
