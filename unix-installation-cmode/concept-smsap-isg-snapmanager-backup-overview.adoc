= SnapManager 备份概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapManager 使用 NetApp Snapshot 技术创建数据库备份。您可以使用 DBVERIFY 实用程序，也可以使用 SnapManager 验证备份的完整性。

SnapManager 通过为包含数据文件，控制文件和归档日志文件的卷创建 Snapshot 副本来备份数据库。这些 Snapshot 副本共同构成了一个备份集， SnapManager 可使用该备份集还原数据库。
